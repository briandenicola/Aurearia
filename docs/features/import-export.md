# Data Import/Export

> Import coins from JSON, export full collection, backup/restore

## Overview

Import, export, and backup tools live under **Settings -> Data**.

## Key Features

- **CSV import** — Download the template, fill collection rows, and import starter data.
- **JSON backup/export** — Export structured collection data for backups or migration.
- **PDF export** — Generate catalog-style documents for insurance, provenance, or sharing.
- **API keys** — Create scoped keys for external tools from the same data/settings area.
- **Safe restores** — Use exports as backups before major edits or self-hosted upgrades.

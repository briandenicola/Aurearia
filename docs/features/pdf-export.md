# PDF Export

> Generate insurance/provenance catalogs with photos and structured data

## Overview

Generate PDF catalog documents from **Settings -> Data** for insurance, provenance packets, or offline review.

## Key Features

- **Collection catalog** — Export coin records with core metadata and images.
- **Insurance support** — Include purchase/current value fields where present.
- **Provenance packet** — Preserve dealer/source, purchase date, notes, and references for records.
- **Private workflow** — PDF generation uses your authenticated collection data.
